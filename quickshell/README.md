# quickshell

This bar started from `tonybanters/quickshell-btw` and was then adapted for this setup.

Credit:
- https://github.com/tonybanters/quickshell-btw#arch-linux

Local changes:
- switched the palette to Catppuccin Frappe
- replaced the bundled image with the system Arch logo
- wired it into Hyprland autostart instead of Waybar
- added notification center and toast animations, loosely inspired by `caelestia-dots/shell`

Animation reference:
- https://github.com/caelestia-dots/shell
