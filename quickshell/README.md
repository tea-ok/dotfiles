# quickshell

This bar started from [`tonybanters/quickshell-btw`](https://github.com/tonybanters/quickshell-btw#arch-linux) and was then adapted for this setup.

Local changes:

- switched the palette to Catppuccin Frappe
- replaced the bundled image with the system Arch logo
- ported the notification popup/center behavior from [`caelestia-dots/shell`](https://github.com/caelestia-dots/shell)
